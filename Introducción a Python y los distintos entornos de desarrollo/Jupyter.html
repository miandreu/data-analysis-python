Jupyter Notebook¶

1.1. Introducción¶

  • Jupyter notebook es una aplicacin que permite el uso integrado de cdigos de programacin junto con presentacin de contenidos (texto, frmulas, figuras, y medios audiovisuales).

  • Facilita una explicacin ms detallada y atractiva de los conceptos que se quieren describir.

  • Los documentos generados se visualizan con un navegador (Explorer, Chrome, Firefox, ...) y pueden incluir cualquier elemento accesible a una pgina web, adems de permitir la ejecucin de cdigo escrito en el lenguaje de programacin Python.

  • Jupyter contiene todas las herramientas cientficas estndar de Python, y permite realizar taras tpicas en el contexto del anlisis de datos: importacin y exportacin, manipulacin y transformacin, visualizacin, etc

1.2. Ejecución¶

Para ejecutar Jupyter hay dos formas:

  • Se busca en la instalacin realizada el icono de "Jupyter Notebook", y se pulsa sobre el mismo.
  • Se abre un terminal y se ejecuta el comando jupyter notebook.

En cualquiera de los casos, aparece una nueva terminal donde se ejecuta la herramienta.

Y se lanza un navegador donde se puede acceder a la interface principal del Jupyter Notebook.

1.3. Configuración del directorio de trabajo¶

Por defecto, Jupyter trabaja en un directorio de trabajo. Sin embargo, es posible crearse un directorio de trabajo propio. Para ello, bien se crea desde la interface el directorio sobre el que se quiere trabajar como se ha visto antes, o bien se abre un terminal, y se navega hasta la carpeta en el que se quiere crear el directorio de trabajo, y all se crea el directorio mkdir directorio . A continuacion se accede al nuevo directorio, y all se ejecuta jupyter notebook

1.4. Los notebooks de Jupyter¶

Los notebooks de Jupyter son unos archivos con extensin .ipyb, en los que se puede escribir cdigo python ejecutable, texto, dibujar grficas y otras operaciones ms.

Para crearse un nuevo notebook basta pulsar sobre el desplegable que aparece en la parte superior derecha denominado New, y seleccionar en el mismo cualquiera de las opciones etiquetadas como Python [conda root] o Python [default] que se muestran debajo de Notebooks.

Una vez pulsado, se carga el nuevo notebook creado

1.5. Elementos de un notebook¶

  • Ttulo del notebook. Cada notebook tiene un ttulo asociado. Por defecto aparece con el nombre Untitled. Para modificarlo, basta pulsar sobre Untitled, y aparece una ventana en la que se puede modificar el nombre Una vez renombrado, aparece el nuevo ttulo en la parte superior del notebook.

  • Mens y Barra de herramientas. En la parte superior de la interface del notebook dispone de un conjunto de mens con diferentes opciones para gestionar los archivos, para editar, visionar,etc Y justo debajo de los mens, aparece un conjunto de iconos que permite realizar las acciones ms comunes sobre los elementos del notebook

  • Celdas: La unidad de edicin de un notebook son las celdas. Cada celda contiene cdigo Python o bien la informacin que documenta dicho cdigo. En este sentido,un notebook es una secuencia de celdas Las celdas pueden ser de diferentes tipos segn el contenido que almacenan:

    • Markdown: Permite escribir texto formateado con el objetivo de documentar. Se usa el lenguaje de marcas Markdown

    • Raw NBConvert: Son celdas que permiten escribir fragmentos de cdigo sin que sean ejecutados.
    • Heading: Permite embeber cdigo html.
    • Code: Sirven para escribir cdigo Python ejecutable. Estn marcadas por la palabra In [n] y estn numeradas. El resultado de la ejecucin de cada celda se muestra en el campo Out[n], tambin numerado. A continuacin se muestra un ejemplo de una celda de cdigo: Como se puede ver en el ejemplo anterior, ambas celdas tienen el mismo nmero, indicando que la segunda es el resultado de la primera. La primera celda In [ ] contiene cdigo a ejecutar y la segunda celda Out[ ] se genera automticamente como resultado de la ejecucin de la primera. Observar que los comentarios dentro de secciones de cdigo se hace poniendo el smbolo de gato, #, al inicio de la lnea.

Para elegir el tipo de celda a utilizar, se selecciona en un desplegable que aparece en la fila superior junto a los iconos.

1.6. Ejecución de las celdas¶

Todas las celdas son susceptibles de ser ejecutadas. La ejecucin de una celda de cdigo Python, ejecutar el cdigo escrito en la celda y producir una salida, que ser el resultado de la ejecucin. La ejecucin de celdas de tipo Markdown, dar formato al texto.

Para ejecutar una celda hay que posicionarse en la celda y posteriormente pulsar el botn cuyo icono es un tringulo mirando a la derecha.De igual forma para parar la ejecucin, se pulsa sobre el cuadrado

Adems de ejecutar las celdas, sobre las celdas se pueden realizar otras acciones de actualizacin:

  • Cortar celdas
  • Pegar celdas
  • Eliminar celdas
  • Cambiar el orden de las celdas

Para crear celdas nuevas se pulsa sobre el botn con el icono del signo +. En la siguiente imagen se pueden ver distintas celdas de texto. La primera celda es de cdigo, la segunda y cuarta son celdas de texto sin ejecutar, mientras que la tercera es una celda de texto ya ejecutada. Si se hace doble click en una celda ya ejecutada, pasa a ser una celda no ejecutada y se podr modificar aadiendo o eliminando texto. A continuacin, se podr volver a ejecutar.

1.7. Flujo de edición de una celda¶

El flujo normal de edicin de una celda consiste en:

  • Elegir el tipo de celda.Por defecto son de tipo Code.Dependiendo del tipo de celda elegido, Jupyter lo interpretar de diferente manera.
  • Una vez introducido el cdigo o texto en la celda,se debe ejecutar.Para ello se pulsa sobre el icono en forma de flecha.
  • A continuacin se genera una nueva celda para editar.

Por ejemplo si se quiere crear una celda que contenga la funcin: def multiplica(a,b): return a*b

Y a continuacion se quiere multiplicar 3 por 4, se hara de la siguiente manera:

  • En primer lugar se crea una celda pulsando sobre el signo + para crear la celda vaca. Como es de tipo Code no hace falta cambiar el tipo de celda en el desplegable:
  • Se copia el texto de la funcin en la celda
  • Se ejecuta la celda pulsando sobre el icono en forma de tringulo hacia la derecha
  • Como resultado de la ejecucin, se ha creado una nueva celda de tipo Code.Se copia la invocacin de la funcin escrita anteriormente para los valores 3 y 4, multiplica(3,4)
  • Se ejecuta la celda pulsando sobre el icono en forma de tringulo hacia la derecha, generndose el resultado de la invocacin.

1.8.5. Exportar e importar los notebooks¶

Una opcin muy interesante de Jupyter es la posibilidad de importar y exportar los notebooks. De esta forma se facilita compartir y reutilizarlos.

Para exportar un notebook, basta tenerlo abierto, y usar el men File.Se selecciona Download as,y dentro de ese desplegable se elige la opcin de exportacin.En general se elegir Notebook(.ipnb). No todas funcionan, pues requieren de tener instaladas librerias adicionales. Para importar un notebook,hay que irse a la pgina principal de Jupyter,dnde aparecen listados todos los notebooks,y basta arrastrar el notebook que se quiere importar. Tambin se puede cargar el notebook, pulsando sobre la opcin upload, de manera que aparece un navegador de archivos, desde donde se elige el archivo que se quiere importar.
